Criar um arquivo .env e preenchendo as variáveis para acessar o database:

```
DB_USER = "<nome_do_usuário>"
DB_PASSWORD = "<senha>"
DB_HOST = "<hot>"
DB_PORT = "<porta>"
DB_NAME = "<nome_do_dataset>"
```
